# Blog perso editorial

Base statique pour un blog personnel avec une direction "zine editorial francais".

## Fichiers

- `index.html` : page d'accueil avec la liste des articles et liens externes
- `styles.css` : direction artistique globale
- `articles/voix-editoriale.html` : exemple d'article long
- `articles/personal-branding.html` : second exemple et gabarit simple

## Personnalisation rapide

1. Remplacer `bonjour@example.com` dans `index.html`.
2. Remplacer les URLs `https://www.linkedin.com/` par tes vrais liens.
3. Changer `--accent` dans `styles.css` si tu veux une autre couleur punch.
4. Dupliquer un fichier dans `articles/` pour creer un nouvel article, puis ajouter sa carte dans `index.html`.

## Direction visuelle

- Fond creme casse avec texture subtile
- Typographie display forte en uppercase
- Metadonnees en monospace
- Layout asymetrique
- Une seule couleur accent

## Lancer localement

Tu peux ouvrir `index.html` directement dans le navigateur, ou lancer un serveur statique depuis ce dossier.
